Template Matching

Template matching is the inverse operation of template expansion. While this operation is not directly defined by RFC 6570, it is useful in verifying and extracting specific information from a URI which is provided in a known format.

Template Matching Basics

Template matching is performed by the Match methods. At this time, the most up-to-date information about the matching operation is provided on that page.
